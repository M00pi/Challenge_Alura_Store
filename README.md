# CHALLENGE ALURA STORE

El siguiente desafío tiene por objetivo entregar un análisis de los datos de venta, categorías, envíos y reseñas, de cuatro Tiendas Alura, para identificar la menos eficiente, con el fin de que el Señor Juan pueda tomar una decisión de cierre y redirigir los recursos a un nuevo emprendimiento.

## Análisis de Datos
El objetivo es analiza la información de facturación para cada tienda, las categorías y productos más y menos vendidas, las reseñas de los clientes y el envío promedio que paga el cliente. Este proceso debe realizarse en el Notebook de Colab utilizando Python y la librería Pandas. 
Para este caso se ha realizado el siguiente análisis: 
- Análisis de facturación total y anual por tienda.
- Ventas por categoría de productos, identificando las 3 categorías con mayor y menor venta por tienda.
- Valoración promedio consolidado por tienda en los 4 años y valoración promedio por tienda por cada año.
- Productos con mayor y menor venta por tienda.
- Envío promedio total y anual por tienda.
- Generar gráficos más relevantes para el análisis. 

## Informe y recomendación
Se genera un informe que resume el trabajo de análisis realizado, muestra los resultados más relevantes, una conclusión y recomendación según lo solicitado por el Señor Juan para la toma de decisiones. 

## Estructura del Proyecto
La carpeta del proyecto contiene las bases de datos de las tiendas Alura, un archivo para ser visualizado en Colab donde se encuentra el desarrollo del proyecto, por último se han guardado las imágenes de los gráficos incluídos en el informe.
